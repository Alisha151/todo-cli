# Todo CLI 📝

A simple Python command-line to-do list manager.  
This project demonstrates clean Python code, Git usage, and basic testing — perfect for beginners learning GitHub workflows.

---

## 🚀 Features
- Add tasks
- List tasks
- Remove tasks
- Lightweight and easy to run

---

## 📦 Requirements
- Python 3.8+
- No external libraries needed

---

Run the program:
```bash
python todo.py
